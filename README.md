### **`temperMonkey脚本`** splunk 快速搜索 


> 该脚本旨在用快捷键代替手动点击splunk的搜索按钮

![splunk搜索](./images/splunk.png)

## Mac
通过键入 `option + r` 来实现 快速搜索

## Windows
通过键入 `Alt + r` 来实现 快速搜索
